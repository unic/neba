NEBA 5.2.1 released

Critical updates to the Spring version shipped with the spring integration

NEBA 5.2.1 ships Spring 5.3.18, which fixes CVE-2022-22965 and CVE-2022-22950. Apart from these critical fixes, the new spring version ships with a number if important updates since version 5.3.15, as per release notes:

  1. Spring v5.3.18
  2. Spring v5.3.17
  3. Spring v5.3.16
